# Dashboard Plan

The dashboard should show analysis, not trading controls.

## Pages

1. Overview
   - watchlist cards
   - final score
   - recommendation
   - risk score
   - confidence

2. Asset Detail
   - market snapshot
   - news feed
   - sentiment explanation
   - risk flags
   - quality flags

3. Signal History
   - saved signals
   - export link
   - filter by symbol

4. Reports
   - JSON report
   - text report
   - latest overview

5. Provider Health
   - exchange mode
   - news mode
   - model adapter status
   - database status
   - streaming status

## Rules

- Do not add real trading buttons.
- Every card must show that output is advisory-only.
- Risk flags must be visible near every recommendation.
