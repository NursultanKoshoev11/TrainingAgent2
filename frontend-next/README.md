# Next Dashboard

Это будущий полноценный dashboard для Crypto AI Advisor.

## Запуск

```bash
cd frontend-next
npm install
NEXT_PUBLIC_API_BASE=http://localhost:8000 npm run dev
```

## Запланированные страницы

- Обзор
- Детальная страница актива
- Статус провайдеров
- Runtime status
- История signals
- Backtest reports

Dashboard должен оставаться advisory-only.
